# Wonder Practices

A reference library of practices for movement, mindfulness, nervous system regulation, and communication.

Live at https://kenyonacton.github.io/wonder-practices.

## Highlights

- Custom SVG category symbols (dot, wave, outline heart, open circle)
- A four-mountain landscape as the persistent category navigation
- Phase-tracking breathing timer with a circle that grows and shrinks with the breath
- Atkinson Hyperlegible body face, self-hosted, designed by the Braille Institute for low-vision readability
- Bricolage Grotesque display face, self-hosted
- Deep-teal palette tuned for legibility in bright and dim viewing
- Categories coded by shape as well as color
- Animations honor system reduced-motion preferences
- Skip link, focus outlines, generous touch targets
- Mobile-first, no backend, no analytics, no third-party scripts

## Built with

Eleventy, plain CSS, plain JS modules. Deployed via GitHub Actions to GitHub Pages.
